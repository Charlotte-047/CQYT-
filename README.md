# CQYT Thesis Formatter 🎓✨

**一键调整论文格式，把毕业论文从“Word 地狱”里捞出来。**

这是一个面向 `.docx` 毕业论文的自动格式整理工具。你把论文丢进去，跑一下脚本，它会尽量帮你把标题、正文、摘要、表格、图片、目录保护等格式问题收拾好，然后再自动检查一遍。

它不是冷冰冰的脚本堆，而是从真实痛点里长出来的小工具：论文格式这事，真的太懂了——不是不会改，是太烦、太碎、太浪费生命。与其让每个毕业生都在 Word 里重复受苦，不如把这些机械劳动交给脚本。

所以它来了：**免费、开源、能跑、能验收，主打一个少熬夜、多毕业。**

---

## 它能干什么？

简单说：**一键调整论文格式。**

它可以帮你处理：

- 标题层级格式；
- 正文字体、字号、缩进、对齐、行距；
- 中文摘要 / 英文摘要；
- 表格格式；
- 图片和媒体保留；
- 页眉页脚、分页等结构检查；
- 源目录保护；
- 格式化后自动验收。

跑完以后，它会告诉你是否通过当前自动检查。不是“看起来差不多”的那种糊弄，而是脚本一项项扫过：标题有没有丢、表格有没有丢、图片有没有丢、长正文有没有丢，能查的都尽量查。

一句话：**让论文格式少折磨人，让毕业生多一点快乐。**

---

## 为什么做它？

因为论文格式真的很会折磨人。

标题、缩进、字号、行距、表格、图片、页码、目录……每一个都看似小事，合起来就是大型精神耐力测试。

尤其是：

- 标题格式一层一个样；
- 正文缩进肉眼看差不多，Word 里完全不是一回事；
- 表格边框调到怀疑鼠标；
- 图片一碰就飞走；
- 目录更是 Word 祖传黑魔法。

这类重复劳动本来就不该靠大家一段一段手改。开发者也是被这些痛点真实暴击过，所以直接跑了一个 skill：把能自动化的格式整理自动化，把毕业生从机械排版里解放一点点。

如果这个工具能帮你少改几个小时、少熬一晚、少骂几句 Word，那它就已经很值了。

## 快速开始

安装依赖：

```bash
pip install lxml
```

格式化论文：

```bash
python scripts/format_paper_xml_only.py "input.docx" "output.docx"
```

检查结果：

```bash
python scripts/verify_all_sections_v1.py "input.docx" "output.docx"
```

如果看到：

```text
VERIFY_ALL_FAILURE_COUNT 0
RESULT 通过
```

说明当前自动验收通过。

## 关于目录

目录是 Word 里最容易炸的地方，所以当前策略很稳：**保护源目录**。

如果你的源文档本来有目录，工具会尽量保留它，不乱改页码、不乱重建、不乱更新。

这不是摆烂，是经验。目录里有字段、制表位、页码、隐藏结构，一通乱改很容易把事情搞复杂。现在的策略就是：能稳稳保住，就别手欠乱动。

## 隐私说明

我们很保护用户隐私，也尊重你的内容权益。

这个仓库不包含真实论文、测试论文内容、个人信息或私有输出文件。工具处理的是你本地提供的文档，不会主动上传、收集或占用你的论文内容。

你的论文是你的，我们只是帮它穿得更整齐一点。

## 使用提醒

这个工具很厉害，但最终提交前还是建议你自己打开 Word 检查一遍。

不是因为它不行，而是论文格式这件事，每个学校、每个老师、每个 Word/WPS 版本都可能有一点小脾气。自动化能帮你省掉大部分机械劳动，最后的确认还是你最靠谱。

建议重点看：

- 目录；
- 摘要；
- 标题；
- 正文；
- 表格；
- 图片；
- 页码；
- 页眉页脚。

放心，它负责冲锋，你负责最后优雅地确认一下。
