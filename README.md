# CQYT Thesis Formatter 🎓✨

这是一个面向移通毕业论文的自动格式整理工具。论文丢进去，跑一下脚本即可

所以它来了：**免费、开源、能跑、能验收**

---

## 它能干啥？

**一键调整论文格式。**

它可以帮你处理：

- 标题层级格式；
- 正文字体、字号、缩进、对齐、行距；
- 中文摘要 / 英文摘要；
- 表格格式；
- 图片和媒体保留；
- 页眉页脚、分页等结构检查；
- 源目录保护；
- 格式化后自动验收。

跑完以后，它会告诉你是否通过当前自动检查。

---

## 快速开始

安装依赖：

```bash
pip install lxml
```

格式化论文：

```bash
python scripts/format_paper_xml_only.py "input.docx" "output.docx"
```

检查结果：

```bash
python scripts/verify_all_sections_v1.py "input.docx" "output.docx"
```

如果看到：

```text
VERIFY_ALL_FAILURE_COUNT 0
RESULT 通过
```

说明当前自动验收通过。

## 使用提醒

论文格式，每个专业、每个老师、每个 Word/WPS 版本都可能有一点小脾气。

自动化能帮你省掉大部分机械劳动，但最终你还是得优雅地确认一下。（因为导师骂的是你😂）

建议重点看：

- 目录；
- 摘要；
- 标题；
- 正文；
- 表格；
- 图片；
- 页码；
- 页眉页脚。

## Fit for ？

适合：

- 正在被论文格式折磨的移通毕业生；
- 想批量处理论文格式的人；
- 想把 Word 重复劳动自动化的人；
- 想少熬夜、多睡觉、顺利毕业的人。

不适合：

- 期待它替你写论文的人；
- 完全不想检查结果的人；
- 纯懒蛋。
